Organizando componentes

O React é composto de componentes, e nos exemplos ja feitos foram criado de forma incorreta a organização, sendo que o certo seria q cada componente seja separado, para q tenha uma organização

O react é um a biblioteca, ele nao te força a nada, mais existem boas maneiras entre os desenvolvedores.

O que é um componente React

É uma classe que tem um metodo render q retorna jsx(componentes de classes) ou uma função q retorna jsx(componentes de função)

Então para organizar os componentes serão repartidos em arquivos
exemplo:

src/components/nomedocomponente/index.jsx

E no exemplo q estamos usando será usada Componentes de função, e será colocado tudo que for do postCard aq neste arquivo index.jsx
