O que é React

React é uma biblioteca javascript para criar interfaces de usuarios, ele é uma biblioteca q será carregada e poderá ser usada em todo o projetou ou em uma determinada parte do projeto.

Ele trabalha com componentes, como menus, carregamentos etc...
O React tem um ecossistema gigante podendo trabalhar em varias areas, como mobile, oculos rift, web etc...
