React Developer Tools

Lembrando que toda a vez que o estado(state) muda o react usa novamente a função render, mais nao é só isso que acontece, pois tambem é passado propriedades(PROPS) para os outros componentes lembrando q esta tudo sendo mudado, isso faz q todos os componentes busquem a usar o metodo Render, ou se for uma função retornará seu componentre jsx.

E para ver mais propriedades do react em seu navegador será preciso instalar a extenção React Developer Tools em seu navegador, ele exibirá as propriedades do seu projeto, como Components
