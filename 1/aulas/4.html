ReactDOM

O pacote react-dom fornece métodos específicos do DOM que podem ser usados ​​no nível superior do seu aplicativo e como uma saída de escape para sair do modelo React, se necessário.

importando
import * as ReactDOM from 'react-dom';
Se você usar ES5 com npm, poderá escrever:
var ReactDOM = require('react-dom');

O pacote react-dom também fornece módulos específicos para aplicativos cliente e servidor:

  • react-dom/client
  • react-dom/server

Visão geral

O pacote react-dom exporta estes, metódos:

  • createPortal()
  • flushSync()
Esses métodos react-dom tambem são exportados, mas são considerados legados:
  • render()
  • hydrate()
  • findDOMNode()
  • unmountComponentAtNote()

Nota:
Ambos render e hydrate foram substituídos por novos métodos de cliente no React 18. Esses métodos avisarão que seu aplicativo se comportará como se estivesse executando o React 17
