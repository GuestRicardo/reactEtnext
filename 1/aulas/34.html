Configuração do .editorconfig

A primeira coisa apos a instalação do React é configurar o .editorconfig para usa-lo é so instalar a extenção, depois com botao direito do mause na raiz do projeto e criar o .editorconfig, ai tem q deixa-lo com a seguinte configuração.

root = true

[*]
indent_style = space
indent_size = 2
end_of_line = lf
charset = utf-8
trim_trailing_whitespace = true
insert_final_newline = true
