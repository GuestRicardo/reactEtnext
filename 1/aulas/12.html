Buscando dados externos com fetch

Documentação FATCH API

É uma forma de adicionar dados externos no componente via API, neste exemplo será usada a API {JSON} Placeholder, e para usa-la lembre-se que será usada dentro do:

componentDidMount() {
}

{JSON} Placeholder é uma API que é usada bastante para buscar dados, e é de graça, ela é para trabalhar com testes pois ela ja tem seus dados ficticios
