Eventos sinteticos do react

É quando você manipula eventos no react, na maioria dos casos estão manipulando eventos sintéticos e que são muito parecido com eventos do DOM

A diferença com os eventos do react com as do DOM

DOM vai precisar de usar o .addEventListener( )
REACT não va precisa, pois internamente o react ja faz isso pra vc.

Exemplos:

Usando eventos DOM geralmente usando em javascript
Usando eventos REACT O grande diferencial é a separação de palavras pela letra maiuscula
