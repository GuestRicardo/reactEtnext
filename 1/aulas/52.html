useDebugValue

useDebugValue é um gancho do React que permite adicionar um rótulo a um gancho personalizado no ReactDevTools

useDebugValue(value, format?)

Referência

useDebugValue(value, format?)

Chame no nível superior do gancho personalizado para exibir um valor de depuração legível: useDebugValue

import { useDebugValue } from 'react';

function useOnlineStatus() {
// ...
useDebugValue(isOnline ? 'Online' : 'Offline');
// ...
}

Parâmetros

  • value: O valor que você deseja exibir no React DevTools. Pode ter qualquer tipo.
  • opcional: Uma função de formatação. Quando o componente é inspecionado, o React DevTools chamará a função de formatação com o como o argumento e, em seguida, exibirá o valor formatado retornado (que pode ter qualquer tipo). Se você não especificar a função de formatação, o próprio original será exibido.formatvaluevalue

Retorna

useDebugValue não devolve nada.

Uso

Adicionando um rótulo a um gancho personalizado
